Python Paste News Blog

I've set up a blog for Python Paste-related news. Interested people can subscribe to the feed.

Feeling expedient, it's just done with Blogger, which uploads files to the server (using a very stripped-down template) and server-side includes that embed that file into the front page. Unfortunately archive pages are probably invalid HTML (since the template's meant to be embedded in another page), but I'm too lazy to deal with that now. This seems like a very easy way to embed news into otherwise-static pages.
