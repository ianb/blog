Web-SIG

There is now a Python Web-SIG. From the charter:
The Python Web SIG is dedicated to improving Python's support for interacting with World Wide Web services and clients.
I would encourage interested people to join the mailing list. My personal agenda is for the Python standard library to include some form of request and response object.
